Getting Started
---------------

Installation (need to upload to PyPI later)::

   pip install -U etk

Example::

   >>> import etk
   >>>

Run ETK CLI::

   pip install -U etk
   python -m etk <command> [options]

For example::

   python -m etk dummy --test "this is a test"
